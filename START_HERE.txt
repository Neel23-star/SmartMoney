❌ "Files weren't created"
✅ Run: node create-all-files.js

❌ "npm install fails"
✅ Make sure you're in backend/ or frontend/ directory
✅ Node.js 16+ is installed: node --version

❌ "Backend won't start"
✅ Port 3001 might be in use
✅ Check: netstat -ano | findstr :3001

❌ "Frontend shows blank page"
✅ Make sure backend is running on http://localhost:3001
✅ Check browser console for errors

❌ "Database errors"
✅ Delete backend/approvals.db
✅ Restart backend (will auto-recreate with fresh data)
